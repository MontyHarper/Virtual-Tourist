# Virtual-Tourist
Fifth assigned project for the Udacity iOS Developer nanodegree.

Virtual Tourist allows you to view photos taken from specific locations all around the world.

Long Press to drop a map pin; photos will download automatically. The app downloads photos as close to the pin location as possible. 

Each pin indicates the number of photos attached to it.

Tap a pin to view its photos.

Long Press a pin to delete it.

Tap the trash can to delete all visible pins.

In the photo album, tap a photo to delete it.

Long press a photo to view it more closely.

Tap the "new page" button to download a new set of photos from the same location.
